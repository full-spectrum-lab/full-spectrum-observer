# Security boundary

This is an Internal Alpha repository bootstrap.

- Network, telemetry, remote models, Connector, and R4 are absent.
- Raw input must not be persisted by default.
- Only `Observer.EngineFacade` may start the private Python worker.
- Dependency and package integrity is enforced by exact versions and SHA-256 manifests.
- Report security issues privately to the project owner; do not include enterprise data.
