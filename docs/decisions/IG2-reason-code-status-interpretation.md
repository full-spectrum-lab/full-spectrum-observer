# IG2 reason-code status interpretation

The frozen source catalog still contains `stability: DRAFT_FOR_VG2_REVIEW`. The
VG2 Design Baseline subsequently approved all 50 codes without changing their
values. IG2 therefore preserves the source file byte-for-byte and records the
effective implementation status in `schemas.lock.json` as
`APPROVED_DESIGN_BASELINE`, under authority `FS-OBS-V010-DES-BL-1.0`.
